## 📦 Deploy It

### ➤ GitHub + Vercel
1. Create a new repo (e.g., `ai-empire-html`)
2. Upload `index.html`
3. Connect to Vercel and deploy (static site)

### ➤ GitHub Pages
1. Push repo to GitHub
2. Go to repo Settings → Pages
3. Set source to `main` branch + `/root`
4. Open your new `*.github.io` page
